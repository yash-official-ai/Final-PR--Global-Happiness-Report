# 🌍 Happiness Score Data Analysis

## 📖 Overview

This project performs **Exploratory Data Analysis (EDA)** on a Happiness Score dataset using **Python, Pandas, Matplotlib, and Seaborn**. The goal is to understand how factors like **GDP, Social Support, Freedom, Healthy Life Expectancy, Generosity, and Corruption** influence a country's Happiness Score.

---

## 🎯 Objectives

- Explore and understand the dataset
- Perform statistical analysis
- Identify the happiest countries
- Visualize key trends
- Draw meaningful insights
